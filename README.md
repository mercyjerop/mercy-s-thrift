# Portfolio Website

## Overview
This is my personal portfolio website built using HTML, CSS, and JavaScript.  
It showcases my skills, projects, and services as a software engineering student at Kisii University.

## Features
- Responsive design for desktop and mobile
- Portfolio section to display projects
- Contact section for clients to reach me
- Demo of a WiFi hotspot login panel (for portfolio purposes)

## How to View
You can view the project by opening the `index.html` file in a browser, or online via GitHub Pages:
